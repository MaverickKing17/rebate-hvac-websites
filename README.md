# GTA HVAC Website Demos

High-conversion, mobile-first website templates and live demos built specifically for **Toronto / GTA HVAC contractors**.

These are ready-to-deploy examples showcasing 2026 design trends, optimized for lead generation in competitive local searches like:

- heat pump installation Toronto
- furnace repair Etobicoke
- no heat emergency service Mississauga
- Ontario rebates for HVAC upgrades

## Features Included in Most Demos

- Modern 2026 UX/UI: mesh gradients, glassmorphic cards, subtle micro-animations, dark mode toggle
- Mobile-first & fast-loading (Core Web Vitals optimized, lazy-loaded WebP images)
- High-conversion elements: sticky CTA bars, rebate savings calculators, urgency timers, testimonials carousels
- AI live chat integration (Tawk.to with custom knowledge base per client)
- White-label ready: easy customization via `lib/config.ts` (company name, logo, colors, phone, service areas, guarantees)
- SEO-friendly: schema markup for LocalBusiness & Services, meta tags, alt text
- Vercel-ready deployment (automatic on push)
